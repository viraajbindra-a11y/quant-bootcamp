# VOID MARKET - Quick Start Guide

## Overview
**VOID MARKET** is the ultimate paper trading platform with 16+ premium features. Renamed from "VOID TERMINAL" with a dark cyberpunk aesthetic.

## File
- **Location:** `/sessions/determined-clever-tesla/mnt/outputs/trading-app.html`
- **Size:** 90 KB (single HTML file, no external dependencies)
- **Browsers:** Chrome, Firefox, Safari, Edge (2023+)

## Top Features

### 1. Real-Time Trading
- Add stocks to watchlist
- Execute Buy/Sell/Short/Cover trades
- Track portfolio performance
- View live data from Yahoo Finance

### 2. Advanced Charting
- **Multi-timeframe:** 1D, 1W, 1M, 3M, 6M, 1Y, 5Y
- **Chart types:** Line & Candlestick toggle
- **Drawing tools:** Trendlines, Support/Resistance lines
- **SMA overlay:** Technical analysis ready

### 3. Options Analysis
- View options chains
- Strike price details
- Call/put pricing
- Volume data

### 4. Smart Tools
- **Position Calculator:** Risk/reward sizing
- **Stock Screener:** Filter by price, sector, cap
- **Price Alerts:** Notifications for target prices
- **Order Book:** Bid/ask depth visualization

### 5. Performance Tracking
- **Trade Journal:** Log reasoning and analysis
- **Analytics:** Win rate, avg gain/loss, Sharpe ratio
- **Leaderboard:** Compete against AI traders
- **Sector Heatmap:** Visual performance grid
- **Economic Calendar:** Market-moving events

### 6. Themes & UI
- **3 Themes:** Light, Dark (default), Midnight
- **Sound Effects:** Buy/sell/alert/success sounds
- **Confetti:** Animation on profitable trades
- **Keyboard Shortcuts:** Press ? to see all

### 7. Data Management
- **Export:** Save portfolio as JSON
- **Import:** Restore from backup
- **LocalStorage:** Persistent data between sessions
- **Timestamp tracking:** All trades recorded

## Getting Started

1. **Open File**
   ```
   Open trading-app.html in your browser
   ```

2. **Add Stocks**
   - Use search box in left sidebar
   - Click to select, watch in real-time

3. **Execute Trade**
   - Enter quantity
   - Click Buy/Sell/Short/Cover
   - Sound effect + confirmation

4. **Track Performance**
   - Right panel tabs show analytics
   - Press ? for all shortcuts
   - Export portfolio anytime

## Troubleshooting

**Data not saving?**
- Check browser's localStorage is enabled
- Try a different browser if needed

**Sound not working?**
- Toggle with 🔊 button in top nav
- Check browser volume settings

**Chart not loading?**
- Ensure stock ticker is valid
- Try a major ticker (AAPL, MSFT, TSLA)

## Support

Features are fully integrated. Use browser console (F12) for debugging if needed.
